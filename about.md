Hiya, I'm Harry 👋

This is my blog / personal site. There isn't really a plan for this project or a stated goal, I've been journalling and reflecting a lot more lately and had the urge to start a blog similar to [Bartosz Ciechanowski](https://ciechanow.ski/). The problem I quickly noticed was that I had nowhere near the breadth or depth of knowledge he has in any particular topic so I thought "why not learn some stuff?", then maybe try to make some explanatory / educational content around that.

The site is hosted on Github Pages, as you probably noticed from the goofy URL, it serves the purpose I need it for. The posts such as the one you're reading now are markdown files written in Obsidian, parsed by Jekyll (which I believe Github Pages is built on in the first place so it's a fitting medium.)

If you want to get in touch, don't. I don't use social media except for YouTube and am purposefully unplugged from things like Instagram, X, Substack etc for my own sanity. Though substack might be worth looking into at some point. Anyway, maybe I'll add a comments feature if I'm feeling brave but don't hold your breath.
