This is the journal — a less-polished space than the main writing. Entries here might be half-thoughts, notes-to-self, or things I want to think about in public.

Each entry has a `published:` flag in its frontmatter. Set it to `true` to make the entry visible here. Leave it off (or set to `false`) to keep it private — the file stays on disk but never appears on the site.
